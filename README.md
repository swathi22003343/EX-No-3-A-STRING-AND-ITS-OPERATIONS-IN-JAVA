## Sourcecode.java:

```
import java.util.Scanner;
public class Main {
	public static void main(String[] args)
	{
       Scanner sc = new Scanner(System.in);
       String str = sc.nextLine();
       System.out.println("The size of "+ "the String is "+ str.length());
	}
}
```
